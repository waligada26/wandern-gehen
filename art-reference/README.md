# art-reference

- **`wanda-reference.png`** — THE master style reference (ART-BIBLE §9):
  the locked 64×64 standing Wanda (G1 candidate C, 11 Jul 2026). Every
  Pixel Lab generation from Section A onward feeds this image as its
  style reference (`style_images` on object tools) and shades toward
  the ART-BIBLE §2 ramps extracted from it.
- **Pixel Lab character id** for Wanda:
  `e3e8a31f-824b-490a-add9-4cc0d2d1e895` — new poses/animations for
  the hiker (Section G's drinking, sitting, picking-up, breathing-idle)
  are `animate_character`/`create_character_state` calls on this id,
  never a fresh generation.
- **`g1-candidates/`** — the four Stage-1 rendering candidates, kept as
  a record of the exploration (A soft, B classic, C winner, D rich).
